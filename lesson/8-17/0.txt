右键-启动
右键-停止
命令启动与关闭
net start/stop  wampstackmysql
服务名称解释
D:\Bitnami\wampstack-5.6.19-0\mysql\bin\mysqld.exe --defaults-file=D:\Bitnami\wampstack-5.6.19-0/mysql\my.ini wampstackMySQL
可以指定-defaults-file 路径加载mysql的启动配置
--defaults-file=D:\Bitnami\wampstack-5.6.19-0/mysql\my.ini

常用命令
查看所有数据库
show databases;
设置整个cli是utf8编码
set names UTF8;
创建数据库
Creat databases php505
使用数据库
 use php505
 创建数据表
 creat table student1（name varchar(40),age int(4)）;
 查看数据库下的所有表
 show tables;
 查看表结构
 desc stutent1;
 查看创建表的sql
 show create table student1;
 插入数据
 insert into student1 (name,age) values('limu',20);
  查询
  select*from student1;

  update student1 set age=12;

  update student1 set age =age+1;

  delete from student1;
  delete student1.*from student1;
  清楚数据
  truncate student1;

  drop table student1;
